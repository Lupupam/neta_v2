History

In mid of 2012, the NETA Framework concept begin to be a real purpose. It is built by the NESG (Network Engineering & Security Group) to provide to the research community a reference and useful tool for simulation and development network attacks.

Currently, in mid of 2013, and architecture and several attacks examples has been created, corroborating the feasibility of this framework as tool to create and to use network attacks in whatever protocol layers and networks.
